# vitepress 更新文档

每次更新 vitepress 文档，都需要执行以下步骤

1. 运行 tags.py 和 categories.py 更新 tags.md 和 categories.md 文件

2. 更新仓库根目录下的 README.md 文件

3. 更新 config.mts 的 siderbar 链接

4. 每个月还得更新 index.md 的 features 列表

每次更新 vitepress 文档，至少有 5 个 git 变动
