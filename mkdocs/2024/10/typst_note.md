# Typst 试用

## 大标题

```typ
= 一级大标题

== 二级大标题
```

## 有序列表

```typ
1. 有序列表
2. 有序列表
+ 有序列表
```

```typ
+ 有序列表
+ 有序列表
```


## 无序列表

```typ
- 无序列表
- 无序列表
```

## 文字

- 字体

```typ
#text(font: "KaiTi", fill: red,size: 140pt, lang: "zh")[红色的楷体，字号为 140pt，语言为中文]
```

- 字体居中

```typ
#set align(center)
```
